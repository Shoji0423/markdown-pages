#### 要素

  - 体力的

  - 精神的
    - 図太い・全然 OK

#### 具体的な内容

- プログラムを触ること

  - コードを書くこと
    - PC 環境・ローカル環境でいじれるやつ
    - 目的がはっきりしているリファクタリング
  - 仮説検証・PoC
    - ライブラリ
    - 意思決定者との距離が近い
      - 判断軸が見える
      - 北極星があると進めやすい

- コミュニケーション(人と話をすること)

  - 情報共有

    - 自分の中で結論が出せたもの
    - 腹落ちした内容
    - 具体例

      - 会話

        - 他人の意見
        - 他人のリアクション
          - 価値観のギャップを把握
        - 関係値構築

      - 記事作成
        - 思考の整理・言語化
          - そのついでに内容を公開
          - 会話のネタになる
      - 登壇
        - そのもの、というよりその後に得られるメリットがある
          - 会話・経験・(人脈)
        - 会話よりも高いアウトプットの質を求められる状況
          - 負荷につながる

  - 1on1
    - 心理的安全性の高くなる要素が大事
    - 関係構築
    - 目的が明確
      - 評価に直結する要素
      - 意思決定者とのすり合わせ

- 自己分析

  - 自分のことは自分が一番理解していたい
  - 自己管理
  - ネガティブ思考
    - 見えない要素を消していきたい
    - なんでも成功するとは考えていない
    - 自己肯定感を積み重ねて、自己の肥大化もしたくない
